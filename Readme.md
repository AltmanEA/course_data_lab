# Сборник задач

Сборник задач по курсу "Прикладное программирование" (обработка данных).

## Решение задач

Для каждой задачи есть тесты, записанные в файле с добавлением суффикса ```.test```. Для запуска теста используйте команду ```npm test <имя задачи>```, например ```npm test collection01```.

Настоятельно рекомендуется решать задачи самостоятельно. При тестировании студентов решение задача будет проверятся в том числе и в ручную. Если при решении задач будут использовать средства, не изучаемые в курсе, или код выглядит сгенерированным ИИ, то задача будет засчитана.

## Задачи, требующие сервера.

При решении задач из блоков mongo и prisma необходимо запустить серверы с помощью ```docker-compose up``` (см. ```docker-compose.yml```). Проверьте настройки подключения к серверам в файлах ```.env``` и ```src/mongo/mongo_init.ts```, по умолчанию строки подключения настроены на работу nodejs в контейнере. При использовании nodejs на ОС замените ```mongo``` и ```postgres``` на ```localhost```.


## Расширение для тестирования студентов

Для запуска тестов установите расширение ```examenator-0.0.1.vsix``` (лежит в корне проекта). Должна появиться панель "Examenator" в боковом меню с тремя вкладками:
- Вкладка "ЗАДАЧИ" показывает блоки и количество задача, при решении задач по ней кликать не нужно.
- На вкладке "ТЕСТЫ" содержит тесты, для запуска нужно кликнуть на соответствующий тест. Это действие открывает задачи, которые нужно решить, на вкладке "АКТИВНЫЙ ТЕСТ" и запускает в строке состояния обратный отсчет времени.
- Вкладка "АКТИВНЫЙ ТЕСТ" показывает список задач текущего теста. Клик по задаче открывает саму задачу, тестовый файл и запускает тест для этой задачи.
